# 4team
## 클래스를 각자 작성 하여 하나의 프로그램 작성하기

2개의 숫자를 사용자에게 입력 받아 아래 메뉴를 선택하여 종료 메뉴를 선택하기 전까지 진행하기

----------------------------------
첫번째 숫자를 입력하세요 : </br>
두번째 숫자를 입력하세요 : </br>
\*\*\*원하는 결과를 선택하세요\*\*\*
1. 덧셈
2. 뺄셈
3. 곱셈
4. 나눗셈
5. 몫과 나머지
6. 종료
----------------------------------

작성 및 사용된 클래스
- Main (Team4_Cal)
- 덧셈 (Cal_add)
- 뺄셈 (Cal_sub)
- 곱셈 (Cal_mul)
- 나눗셈 (Cal_divide)
- 몫과 나머지 (Cal_reminder)
